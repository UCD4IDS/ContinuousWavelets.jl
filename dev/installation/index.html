Installation

At the moment this package is unregistered, so you will have to do it the hard way. Either press ] and run

(@v9.9) pkg> add https://github.com/dsweber2/ContinuousWavelets.jl.git

or download via git e.g. git clone https://github.com/dsweber2/ContinuousWavelets.jl.git.
